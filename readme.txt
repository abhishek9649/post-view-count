Track views for selected posts and pages. Displays post view counts along with user info, login status, date, and post status.

== Description ==

**Post View Count** is a simple and powerful plugin that tracks:
- Total views per post/page
- Username (if logged in)
- Whether the viewer was logged in or not
- Post/page status
- View date and time

You can monitor and manage all tracked data from the WordPress admin panel under the **Post Views** menu.

== Installation ==

1. Upload the `post-view-count` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** menu in WordPress.
3. Visit any post or page â€“ views will start being tracked.
4. Go to **Post Views** in the admin dashboard to view logged data.

== Frequently Asked Questions ==

= Does this plugin track guest users? =  
Yes, it tracks both logged-in and guest users.

= Where is the data stored? =  
It creates a custom table in your database: `wp_post_view_count`.

= Can I export the data? =  
Not yet. Export functionality will be added in a future version.
